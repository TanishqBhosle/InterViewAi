##  Pattern Recognition Guide

### Hashing
**Signals:** "Two sum", "frequency", "duplicate", "group"
**Template:** HashMap for O(1) lookup

### Two Pointers
**Signals:** "Sorted array", "pair", "opposite ends", "palindrome"
**Template:** Left and right pointers moving inward

### Sliding Window
**Signals:** "Substring", "contiguous", "window size", "max/min"
**Template:** Expand right, contract left

### Binary Search
**Signals:** "Sorted", "search", "O(log N)", "minimize max"
**Template:** left + (right - left) / 2

### Dynamic Programming
**Signals:** "Optimize", "count ways", "subsequence", "overlapping"
**Template:** dp[i] = f(dp[i-1], dp[i-2], ...)

### Backtracking
**Signals:** "All solutions", "permutation", "combination", "N-Queen"
**Template:** Choose → Explore → Unchoose
